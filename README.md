# Meus Sites Favoritos - Agenda 12 - Desenvolvimento de Sistemas I

Este projeto foi desenvolvido como atividade da **Agenda 12 - Desenvolvimento de Sistemas I**.

## Objetivo

Criar uma página web em HTML para apresentar cinco sites favoritos, contendo:

- título da aba do navegador;
- título principal no corpo da página;
- nome de cada site;
- uma imagem para cada site;
- link de acesso abaixo de cada imagem;
- imagens também clicáveis, direcionando para os respectivos sites;
- comentários no código explicando as principais partes da página;
- organização visual com CSS.

## Sites escolhidos

1. Google  
   https://www.google.com

2. YouTube  
   https://www.youtube.com

3. GitHub  
   https://github.com

4. Python  
   https://www.python.org

5. W3Schools  
   https://www.w3schools.com

## Como executar

1. Baixe ou clone este repositório.
2. Abra o arquivo `jefferson_astuti_magalhaes_de_barros_ag12_ds_i.html` em qualquer navegador.
3. Verifique se as imagens aparecem corretamente e se os links abrem os sites em uma nova aba.

## Observação

As imagens utilizadas neste projeto são ilustrações simples criadas para fins acadêmicos, representando visualmente os sites escolhidos.
